Welcome to a free Multi-User BibTeX Manager

This website is currently being developed and as a result it may not be functioning properly. Please feel free to try it out but I would not recommend using it as your primary bibliography manager, yet ;).

The website is currently being evaluated and documented and will have an official launch date later this year.

If you have any questions or concerns please contact me and I will get back to you as soon as possible.

Best regards,

Viktor Eriksson


Using this application you will be able to:
  • chevron_rightAdd, Edit, Delete, View your BibTeX records
  • chevron_rightImport bibliographies from file (currently supports .bib files)
  • chevron_rightExport bibliographies to file (currently supports .bib files)
  • chevron_rightfilter, search and sort through big bibliographies
  • chevron_rightInvite friends or collegues to work on your bibliography
  • chevron_rightGive your collegues different level of access (ReadOnly / ReadWrite / Admin)

Sign up here. If you wish to learn more about BibTeX visit www.bibtex.org.
